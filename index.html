BYU's next generation netlist tool

SpyDrNet 1.0.0 is released December 18 2019. This tool aims to help end users more easily manipulate netlists

The framework provides an easy to use api for accessing generic netlist datastructures to analyze them. Our goal in developing this tool was to help promte reliability research by allowing users to easily gain access to netlist datastructures for modification and analysis.
